About the App

This application is designed to track professional match data across esports titles such as CSGO and Valorant. You can browse games, view players, explore match results, and analyze detailed player statistics through a clean, unified interface.

Every page is connected using an intuitive navigation bar, allowing quick access to:

  • All tracked games
  • Professional players
  • Match results
  • Fantasy

The long-term goal is to allow users to draft teams, compete, score points based on player performance, and climb leaderboards.

This is just the beginning, and the project will keep expanding with new features, improved UI, and richer esports data.
